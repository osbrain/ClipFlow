# 详情操作按钮与滚动条视觉设计

## 目标

修复详情区按钮高度跨度过大、同级按钮尺寸不一致，以及滚动条视觉过粗、对比度过高的问题。

## 按钮层级

- 主操作：42pt，高亮强调色填充，连续圆角 12pt。
- 次操作：36pt，中性浅填充和细边框，连续圆角 10pt。
- 收藏、重命名、删除：30pt，同一行等宽，连续圆角 8pt。
- 所有按钮使用统一 14pt 图标、8pt 图文间距和单行缩放文本。
- 飞书、豆包等应用操作使用次操作规格。
- 按下状态仅轻微降低透明度并缩放，避免按钮跳动。

## 滚动条

- 保留 overlay 和自动隐藏行为。
- 使用自定义 `NSScroller` 只绘制 4pt 宽的圆角滑块，不绘制轨道。
- 默认颜色取系统次级标签色，透明度 0.26；鼠标悬停或按下时提高到 0.46。
- 垂直和水平滚动条使用同一几何计算，适配浅色和深色模式。

## 验收

- 自动测试固定 42/36/30pt 按钮指标和 4pt 滚动条厚度。
- 自动测试垂直、水平滑块的缩窄矩形。
- 视觉验收覆盖文字、图片、链接、文件详情以及设置窗口。
- Release `.app` 重新打包并启动。
