## 1.0.3 - 2026-07-16

- Improved semantic deduplication for repeated text, rich text, links, files, images, and fallback binary clipboard captures.
- Reduced history-list work by bounding default loads, batching category lookup, and refreshing duplicate captures incrementally.
- Made relative time labels static instead of continuously updating every visible row.
- Added smoke coverage for 1,000-item, 10,000-item, and repeated-copy clipboard performance paths.
- Updated product documentation and release screenshots for the current Chinese UI, onboarding guide, settings window, and type-specific actions.
