# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately to [aiesst.labs@gmail.com](mailto:aiesst.labs@gmail.com). Include the affected version, a concise reproduction path, and the expected and actual behavior.

Do not open a public issue for a security vulnerability. In particular, do not include clipboard contents, database files, encryption keys, screenshots containing sensitive text, or other personal data in an issue or report unless they are strictly necessary and have been safely redacted.

## Scope

Reports are welcome for the ClipFlow app, its local storage, encryption, packaging scripts, and browser automation integration. ClipFlow is a local-first application: it does not operate a hosted service or accept clipboard data on a server.
